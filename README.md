# 👕 ReWear

A web-based platform that enables users to exchange unused clothing through direct swaps or a point-based redemption system.

---

## 🚀 Features

- 🔁 **Direct Swap System** – Users can list and swap clothes with others directly.
- **Point-Based Redemption** – Earn points by giving away clothes, and redeem them to get items listed by others.
- 📸 **Item Listing with Images** – Upload clothing images, descriptions, sizes, and categories.
- 🧾 **User Dashboard** – View your uploaded items, incoming swap requests, accepted swaps, and point balance.
- 🔐 **Authentication** – User sign-up, login, and session management.
- 📨 **Swap Requests** – Send and receive requests to exchange clothes.
- 🧠 **Inventory Management** – Add, edit, or delete your clothing listings.
